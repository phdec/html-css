O que é um Menu Flexibe?

Um Menu Flexible (geralmente criado com CSS Flexbox) é uma estrutura de navegação responsiva que organiza itens automaticamente, ajustando-se a qualquer tamanho de tela. Ele permite alinhamento horizontal/vertical, espaçamento uniforme e quebra de linha automática (wrap), ideal para layouts modernos que precisam funcionar em dispositivos móveis e desktop.
